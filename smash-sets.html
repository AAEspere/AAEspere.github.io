Smash Sets

Smash Sets was a small side project I created that allows users to look up tournament sets for the game Super Smash Bros. Melee (melee) by Nintendo. I have a interest in the game, and decided to create this after having a hard time looking a sets for a certain character - since there are 5 games, search results can be a mix of all 5 of these games. Smash Sets uses the melee character select screen as its UI. Users just have to click on one of the characters, and the application will return tournament sets in the bottom which link to their respective Youtube videos.

This app has a HTML/CSS/JavaScript front end and a NodeJS runtime environment. It uses the YouTube API to search and return results. I plan to further expand on this project in the future. First, I would probably integrate a database (thinking MySQL) into the application. Having a database will allow me to do more searches without reaching the limit usage of the YouTube API. Second, I would probably add features to search sets based on player and tournament. Finally, once all features are implemented, I would probably have the results show on a modal that is shown after the character is chosen.
